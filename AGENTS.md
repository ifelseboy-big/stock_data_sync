# 项目操作规则

- 远程部署机器统一使用 SSH 别名 `lingfeng-local` 连接，不直接使用 IP 地址或其他用户名。
- 远程 Web 入口固定为 `http://192.168.2.140:8888/`；Web 与服务 API 共同监听 `192.168.2.140:8888`，API 前缀为 `/api/v1`，不得省略端口或改用 `127.0.0.1` 猜测监听地址。
- 远程主程序目录：`/Users/lingfeng/personal_apps/stock_data_sync`。
- 远程数据目录：`/Volumes/disk1/apps/stock_data_sync`。
- 排障默认只执行只读检查；修改配置、数据、服务状态或部署版本前，必须获得用户明确授权。
- 数据库结构改动必须同步更新 ORM 表结构、可从空库执行到 `head` 的 Alembic 迁移、升级/全新安装验证及数据模型和部署文档，保证新机器首次部署后的数据库结构与当前目标结构完全一致，已有数据库升级后也收敛到同一结构；涉及重大业务或技术流程调整时，必须同步更新描述当前实现的技术文档，不得只修改代码、测试或历史计划文档。
