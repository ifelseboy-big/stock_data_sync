# 接口目录与原始资产

状态：已完成

## 目标

集中声明接口和输出数据集规则，并实现不可变 Parquet 原始资产存储，为采集与加工解耦提供稳定契约。

## 实现任务

1. 增加 PyArrow 并锁定依赖版本。
2. 实现 `ApiSpec`：字段、阶段、范围、拆分、行数上限、空结果、重试和业务日期提取。
3. 实现 `DatasetSpec`：依赖、处理器版本、写入策略、发布范围和质量规则。
4. 实现 `RawAssetStore` 协议与 `LocalRawAssetStore`。
5. 完成流式写入、Zstandard 压缩、结构指纹、SHA-256、`fsync` 和原子改名。
6. 实现零行合法资产、孤儿文件检查和路径越界防护。

## 验收条件

- 一个采集任务只产生一个不可变资产。
- 分页结果不要求一次性驻留内存。
- 失败写入不留下可被解析为 READY 的最终文件。
- 相同字段顺序得到稳定结构指纹，字段变化能够被识别。
